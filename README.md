# simpleChatServer
------------------

Real Time webChat server with NodeJS, Socket.io, ExpressJS and Jade as the template engine.

#### Install & run

```
git clone https://github.com/Anirban2404/simpleChatServer.git && cd simpleChatServer && npm install
```

```
node index.js
```
or
```
npm start
```
